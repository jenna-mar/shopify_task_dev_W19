createLineItem (LineItem!)

Create a new line item. Creating a new line item will also add its total cost to its order. The order and product IDs given must be from the same shop.

Argument Type Description
quantity Int!

The quantity of the item purchased in the order.
orderId ID!

The ID of the order that the line item is part of.
productId ID!

The ID of the product that the line item represents.
editLineItem (LineItem!)

Edit a line item. If the line item is a given a new product ID, the title and price will be updated to match. Otherwise, the provided title and price will take precedence, but the product ID will still be updated if present. The order total will also be adjusted to reflect changes in price, if applicable. The order and product IDs must be from the same shop.

Argument Type Description
id ID!

The ID of the line item to be edited.
title String

The new title name of the line item.
quantity Int

The new quantity of the line item.
price Float

The new price of the line item.
orderId ID

The new order that the line item is part of.
productId ID

The new product that the line item represents.
deleteLineItem (LineItem!)

Delete a line item. The value of the line item will be removed from its order (total price) upon deletion.

Argument Type Description
id ID!

The ID of the line item to be deleted.
