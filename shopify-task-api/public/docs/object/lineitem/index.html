LineItem

A line item in an order.

Fields

id (ID!)
title (String!)

The name of the line item. This is the same as the title of its product on creation.

quantity (Int!)

The quantity of product purchased in the order.

price (Float)

Price of the line item. When a line item is created, it is given the price of its product.

order (Order!)

The line item's order.

product (Product!)

The line item's product.
